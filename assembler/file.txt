assembler.py программа по сути сам ассемблер он выдает на выход my_filename.hack файл это двоичное представление инструкций процессора

my_filename.hack (my_add.hack к примеру) из перой строчки понятно че такое

filename.hack (add.hack к примеру) это файл уже заранее заготовленный то есть это правильное двоичное представление инструкций процессора
он тут нужен чтобы сверять filename.hack и my_filename.hack чтобы понять совпрадают ли они оба правильно ли сработал assembler.py

filename.asm (add.asm к примеру) это ассемблер файл код который идет на вход программе assembler.py

diff.py программа которая сверяет оба файла my_filename.hack и filename.hack
